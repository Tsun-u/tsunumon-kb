# tsunumon-kb

Independently compiled study-reference notes used as a knowledge base for an
educational AI tutoring project. The material is organized by subject
(Biology, Computer Science, Mathematics, Physics) as a quick conceptual
reference — definitions, worked-example outlines, common misconceptions, and
diagram/tool API notes.
