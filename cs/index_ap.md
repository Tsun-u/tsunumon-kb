# AP® Computer Science A - Course Framework

## Course Overview

AP Computer Science A is a college-level programming course that uses Java as its primary language. Students build practical coding skills by reading, writing, and testing programs that demonstrate fundamental concepts including data types, control flow, object-oriented design, arrays, and algorithm development.

**Language:** Java

## Important: 2025-26 Curriculum Revision

Starting with the 2025-26 school year, College Board restructured the course from **10 units to 4 units**:

| New Unit | Exam Weight | Covers (Old Units) |
|----------|-------------|---------------------|
| Unit 1: Using Objects and Methods | 15-25% | Old Units 1, 2 |
| Unit 2: Selection and Iteration | 25-35% | Old Units 3, 4 |
| Unit 3: Class Creation | 10-18% | Old Unit 5 |
| Unit 4: Data Collections | 30-40% | Old Units 6, 7, 8 |

**Key Changes:**
- Inheritance (old Unit 9) and polymorphism **removed** from required content
- Recursion (old Unit 10) **removed** from required content
- Text file I/O and dataset manipulation **added** (File and Scanner classes)
- Emphasis on data-driven programming tasks

The files in this directory document the **original 10-unit framework** which many teaching resources still reference, with notes on the revised structure where applicable.

## Computational Thinking Practices

| Practice | Description | Exam Weight |
|----------|-------------|-------------|
| 1. Design Code | Plan programs and craft algorithmic solutions to problems | 2-10% |
| 2. Develop Code | Write and run Java programs from specifications | 22-38% |
| 3. Analyze Code | Read and trace code to predict outputs and behaviors | 37-53% |
| 4. Document Code | Explain program intent, conditions, and expected results | 10-15% |
| 5. Use Computers Responsibly | Recognize the ethical dimensions of computing decisions | 2-10% |

## Big Ideas

| Big Idea | Code | Description |
|----------|------|-------------|
| Modularity | MOD | Structuring programs into reusable, well-scoped components |
| Variables | VAR | Representing and transforming data with named memory locations |
| Control | CON | Shaping program execution through conditionals and loops |
| Impact of Computing | IOC | Understanding how software choices carry ethical and social weight |

## Exam Format

- **Section I:** 40 Multiple Choice Questions (50% of score, 90 minutes)
- **Section II:** 4 Free Response Questions (50% of score, 90 minutes)
  - FRQ 1: Methods and Control Structures
  - FRQ 2: Classes
  - FRQ 3: Array/ArrayList
  - FRQ 4: 2D Array

## Java Subset

The AP CS A exam uses a subset of Java. Key classes/methods students must know:
- `System.out.print()`, `System.out.println()`
- `String` methods: `length()`, `substring()`, `indexOf()`, `equals()`, `compareTo()`
- `Math` class: `abs()`, `pow()`, `sqrt()`, `random()`
- `Integer` class: `Integer.MIN_VALUE`, `Integer.MAX_VALUE`
- `Double` class
- `ArrayList` methods: `add()`, `get()`, `set()`, `remove()`, `size()`
- Array: `length` field, indexing with `[]`
