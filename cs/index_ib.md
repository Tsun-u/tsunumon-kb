## New Syllabus (First Teaching 2025, First Assessment 2027)

The upcoming syllabus restructures topics into **two themes**:

| Theme | Name | Focus |
|-------|------|-------|
| **A** | How Computing Systems Work | Hardware, networks, databases, machine learning |
| **B** | Solving Problems Using Computing | Computational thinking, programming, OOP, ADTs |

### Theme A Topics
- A.1 Computer Fundamentals (hardware, software, system architecture, ethics)
- A.2 Networks (protocols, cybersecurity, cloud computing)
- A.3 Databases (SQL, relational models, data storage)
- A.4 Machine Learning (new topic)

### Theme B Topics
- B.1 Computational Thinking
- B.2 Programming
- B.3 Object-Oriented Programming
- B.4 Abstract Data Types (HL only)

## Key Differences from AP Computer Science

- IB CS covers **broader topics** (networking, databases, control systems) vs. AP CS A's focus on Java programming
- AP CS A is purely **programming-focused** (Java); IB CS includes significant **theory** (system design, networking)
- IB CS HL covers **abstract data structures** (linked lists, trees, stacks, queues) similar to a university data structures course
- AP CS Principles is closer to IB CS SL in breadth but with less programming depth
- IB CS includes a **case study** component; AP does not
- IB CS requires an **Internal Assessment** (computational solution for a real client); AP has no equivalent
- The new IB syllabus (2027) adds **machine learning** and **databases** as explicit topics

## Usage Notes for AI Teaching Agent

- Always specify SL vs. HL content boundaries
- Use topic numbers (e.g., Topic 3.1.6) when referencing specific content
- IB CS emphasizes **system design** and **real-world application** more than AP
- Programming examples should be in **Java** (IB's officially supported language) or pseudocode
- The case study changes annually; focus on transferable concepts rather than case-study-specific details
- For the new syllabus (2027+), note the addition of machine learning and databases as core content
