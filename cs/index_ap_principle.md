# AP Computer Science Principles - Course Framework

## Course Overview

AP Computer Science Principles is an introductory college-level computing course that introduces students to the breadth of the field of computer science. Students learn to design and evaluate solutions, apply computer science to solve problems through algorithms and programs, incorporate abstraction, use data to discover new knowledge, and explain how computing innovations and systems work while exploring their potential impacts.

**Language:** Language-agnostic (many implementations use Python, JavaScript, or block-based languages)

## Exam Format

### End-of-Course Exam (70% of AP Score)
- **Duration:** 2 hours (120 minutes)
- **70 multiple-choice questions:**
  - 57 single-select (choose 1 from 4)
  - 5 single-select with reading passage about a computing innovation
  - 8 multiple-select (choose 2 from 4)
- Written response questions related to the Create Performance Task code

### Create Performance Task (30% of AP Score)
- Students develop a program of their choice
- Minimum 9 hours of dedicated in-class time
- Students create a Personalized Project Reference (program code + video)
- On exam day, students respond to written-response questions about their code

## Key Differences from AP CS A

| Aspect | AP CS Principles | AP CS A |
|--------|-----------------|---------|
| Focus | Breadth of CS | Java programming depth |
| Language | Any language | Java only |
| Programming | Introductory | Intermediate |
| Topics | Internet, data, impact | OOP, data structures |
| Performance Task | Yes (Create PT) | No |
| Difficulty | Generally easier | More rigorous |

## Common Teaching Unit Structures

While the College Board framework is organized by Big Ideas, many curricula (such as Code.org) organize teaching into sequential units:

1. Digital Information (Binary, data representation)
2. The Internet (Networks, protocols)
3. Programming Fundamentals (Variables, conditionals)
4. Programming with Lists and Loops
5. Algorithms and Data
6. Create Performance Task
7. Data Analysis and Visualization
8. Innovation and Impact
9. Cybersecurity
10. Review and Exam Prep
