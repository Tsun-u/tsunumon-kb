# A.5: Galilean and Special Relativity

> Theme A: Space, Time and Motion | HL only

## Key Concepts

### Galilean Relativity
- Galilean transformation: $x' = x - vt$, $t' = t$
- Velocity addition: $u' = u - v$
- This framework accurately describes motion when speeds are far below $c$
- The laws governing mechanical phenomena look identical in every inertial reference frame

### Einstein's Postulates of Special Relativity
1. The laws of physics are the same in all inertial reference frames
2. The speed of light in a vacuum ($c = 3.0 \times 10^8$ m/s) is the same for all observers

### Consequences of Special Relativity

#### Time Dilation
- A clock in motion relative to an observer ticks more slowly than a stationary one from that observer's perspective
- $\Delta t = \gamma \Delta t_0$ where $\gamma = \frac{1}{\sqrt{1 - v^2/c^2}}$
- $\Delta t_0$: proper time (measured in the frame where events occur at the same location)
- Evidence: muon decay experiments, GPS satellite corrections

#### Length Contraction
- An object moving relative to an observer appears shortened along its direction of travel
- $L = \frac{L_0}{\gamma}$
- $L_0$: proper length (measured in the object's rest frame)

#### Relativistic Momentum
- $p = \gamma mv$
- At speeds much less than $c$, $\gamma \approx 1$ and this expression recovers the classical result $p = mv$

#### Mass-Energy Equivalence
- Rest energy: $E_0 = mc^2$
- Total energy: $E = \gamma mc^2$
- Kinetic energy: $E_k = (\gamma - 1)mc^2$
- Energy-momentum relation: $E^2 = (pc)^2 + (mc^2)^2$

### The Lorentz Factor
- $\gamma = \frac{1}{\sqrt{1 - v^2/c^2}}$
- $\gamma = 1$ when $v = 0$; $\gamma \to \infty$ as $v \to c$, which means no massive object can ever reach the speed of light

### Spacetime Diagrams
- Each object traces a worldline—its complete path through both space and time plotted on a single diagram
- Light cones mark the boundary of events that can causally influence each other
- Two events that are simultaneous in one inertial frame will generally not appear simultaneous in another

## Key Equations

| Equation | Description |
|----------|-------------|
| $\gamma = \frac{1}{\sqrt{1-v^2/c^2}}$ | Lorentz factor |
| $\Delta t = \gamma \Delta t_0$ | Time dilation |
| $L = L_0 / \gamma$ | Length contraction |
| $p = \gamma mv$ | Relativistic momentum |
| $E_0 = mc^2$ | Rest energy |
| $E = \gamma mc^2$ | Total relativistic energy |
| $E^2 = (pc)^2 + (m_0 c^2)^2$ | Energy-momentum relation |

## Common Misconceptions

1. **"Nothing can travel faster than light"** -- More precisely, nothing with mass can be accelerated to or beyond the speed of light. The expansion of space itself can exceed $c$.
2. **"Time dilation is an illusion"** -- Time dilation is a real, measurable effect confirmed by experiments with atomic clocks and muon lifetimes.
3. **"Objects physically shrink due to length contraction"** -- Length contraction is a measurement effect between frames, not a physical compression.
4. **"Mass increases with velocity"** -- Modern physics prefers to say momentum increases relativistically; rest mass is invariant.
