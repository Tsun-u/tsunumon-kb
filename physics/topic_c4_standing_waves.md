# C.4: Standing Waves and Resonance

> Theme C: Wave Behaviour | SL + HL

## Key Concepts

### Formation of Standing Waves
- Standing waves arise when two waves of the same frequency and amplitude travel in opposite directions through the same medium and their superposition produces a fixed spatial pattern
- **Nodes**: locations where the two waves always cancel, leaving the medium permanently at rest
- **Antinodes**: locations where the two waves always reinforce, producing the largest oscillation in the pattern
- Adjacent nodes are separated by exactly $\lambda/2$
- Every point of the medium between a pair of adjacent nodes oscillates with the same phase

### Boundary Conditions
- **Fixed ends** (strings, closed pipes): the boundary forces a node
- **Open ends** (open pipes): the boundary allows an antinode

### Harmonics on Strings (Fixed at Both Ends)
- Fundamental (1st harmonic): $L = \frac{\lambda_1}{2}$ → $f_1 = \frac{v}{2L}$
- 2nd harmonic: $L = \lambda_2$ → $f_2 = 2f_1$
- $n$th harmonic: $f_n = n \cdot f_1 = \frac{nv}{2L}$
- Speed on a string: $v = \sqrt{\frac{T}{\mu}}$ (tension / linear mass density)

### Harmonics in Pipes
- **Open pipe** (open both ends):
  - Fundamental: $L = \frac{\lambda_1}{2}$ → $f_1 = \frac{v}{2L}$
  - The full sequence of harmonics is supported: $f_n = nf_1$
- **Closed pipe** (closed one end):
  - Fundamental: $L = \frac{\lambda_1}{4}$ → $f_1 = \frac{v}{4L}$
  - Only the odd-numbered harmonics fit the boundary conditions: $f_n = nf_1$ where $n = 1, 3, 5, ...$

### Resonance
- A standing wave establishes itself only at discrete frequencies that satisfy the boundary conditions; these are the resonant frequencies
- When driven at one of these frequencies, the system builds up oscillation to its maximum possible amplitude
- The rich tone of musical instruments comes from multiple harmonics excited simultaneously through standing-wave resonance

## Key Equations

| Equation | Description |
|----------|-------------|
| $f_n = \frac{nv}{2L}$ | Harmonics on string / open pipe |
| $f_n = \frac{nv}{4L}$ (odd $n$) | Harmonics in closed pipe |
| $v = \sqrt{T/\mu}$ | Wave speed on a string |
| $f_1 = \frac{v}{2L}$ | Fundamental frequency (string/open pipe) |

## Common Misconceptions

1. **"Standing waves don't transfer energy"** -- Correct! This is a key distinction from traveling waves; standing waves store energy.
2. **"Closed pipes have all harmonics"** -- Closed pipes (one closed end) only produce odd harmonics.
3. **"Nodes move"** -- Nodes are fixed positions; the wave pattern does not travel.
4. **"Higher harmonics are louder"** -- Typically, higher harmonics have lower amplitude; the fundamental is usually strongest.
