# C.5: Doppler Effect

> Theme C: Wave Behaviour | SL + HL

## Key Concepts

### The Doppler Effect
- Change in observed frequency (and wavelength) due to relative motion between source and observer
- Source approaching: higher frequency (shorter wavelength) observed
- Source receding: lower frequency (longer wavelength) observed

### Doppler Effect for Sound
- Moving source: $f' = f \cdot \frac{v}{v \mp v_s}$
  - $-$ when source approaches (frequency increases)
  - $+$ when source recedes (frequency decreases)
- Moving observer: $f' = f \cdot \frac{v \pm v_o}{v}$
  - $+$ when observer approaches
  - $-$ when observer recedes
- $v$: speed of sound in medium; $v_s$: source speed; $v_o$: observer speed

### Doppler Effect for Light
- For electromagnetic waves (no medium required):
- $\frac{\Delta f}{f} = \frac{\Delta\lambda}{\lambda} \approx \frac{v}{c}$ (for $v \ll c$)
- **Red shift**: source moving away → wavelength increases (shifted toward red)
- **Blue shift**: source moving toward → wavelength decreases (shifted toward blue)

### Applications
- **Astronomy**: red shift of distant galaxies → evidence for expanding universe
  - Hubble's law: $v = H_0 d$ (recession velocity proportional to distance)
- **Medical**: Doppler ultrasound for blood flow measurement
- **Radar**: speed detection (police radar guns)
- **Sonar**: underwater detection and mapping

## Key Equations

| Equation | Description |
|----------|-------------|
| $f' = f\frac{v}{v \mp v_s}$ | Doppler (moving source) |
| $f' = f\frac{v \pm v_o}{v}$ | Doppler (moving observer) |
| $\frac{\Delta f}{f} \approx \frac{v}{c}$ | Doppler for EM waves ($v \ll c$) |
| $v = H_0 d$ | Hubble's law |

## Common Misconceptions

1. **"The Doppler effect changes the actual frequency emitted"** -- The source frequency does not change; only the observed frequency changes.
2. **"The Doppler effect only works for sound"** -- It applies to all waves, including light and other EM radiation.
3. **"Red shift means the object is red"** -- Red shift means wavelengths are shifted toward the red end; the object may not appear red.
4. **"The Doppler effect requires the source to move"** -- The effect occurs with relative motion; the observer can be moving instead.
