# Phosphor Icons — Verified Name Reference

**CRITICAL RULE: use ONLY icon names from this page.** Any name not in the
locally-installed Phosphor version renders as a BLANK SPACE in the slide
(no error, no fallback). Do NOT guess names from memory — e.g. `ph-telescope`,
`ph-loop`, `ph-circuit-board`, `ph-map`, `ph-reflect-horizontal` all LOOK
plausible but DO NOT EXIST here and rendered as blanks in past lessons.

## Usage

```html
<i class="ph-bold ph-{name}"></i>   <!-- bold weight (phase badges) -->
<i class="ph ph-{name}"></i>        <!-- regular weight (inline icons) -->
```

Only `ph` (regular) and `ph-bold` weights are installed. Do not use
`ph-fill` / `ph-duotone` / `ph-light` / `ph-thin`.

If the icon you want is not listed below, pick the closest match or a
generic fallback from the last section — never invent a name.
