# D1: Continuity and Change -- Molecules

> Theme D: Continuity and Change | Level 1: Molecules

## Subtopics

### D1.1 DNA Replication (SL + HL)

- **Semi-conservative replication**: each new DNA molecule has one original and one new strand
  - Meselson-Stahl experiment: evidence for semi-conservative replication
- Replication process:
  - **Helicase** unwinds and separates the double helix at the replication fork
  - **DNA polymerase III** synthesizes new strands in the 5' → 3' direction
  - **Leading strand**: continuous synthesis toward the replication fork
  - **Lagging strand**: synthesized in Okazaki fragments away from fork
  - **DNA ligase** joins Okazaki fragments
  - **Primase** adds RNA primers to initiate synthesis
- Replication is bidirectional from multiple origins in eukaryotes
- Error rate and proofreading by DNA polymerase

### D1.2 Protein Synthesis (SL + HL)

- **Central dogma**: DNA → (transcription) → mRNA → (translation) → protein
- **Transcription** (in nucleus):
  - RNA polymerase binds to promoter and separates DNA strands
  - mRNA synthesized 5' → 3' using antisense (template) strand
  - Complementary base pairing: A-U, T-A, G-C, C-G
  - In eukaryotes: pre-mRNA processing (5' cap, 3' poly-A tail, intron splicing)
- **Translation** (on ribosomes):
  - mRNA codons read 5' → 3' by ribosome
  - tRNA anticodons match mRNA codons; carry specific amino acids
  - Start codon: AUG (methionine); stop codons: UAA, UAG, UGA
  - Peptide bonds form between amino acids → polypeptide chain
- The genetic code is universal, degenerate (redundant), and non-overlapping

### D1.3 Mutation and Gene Editing (SL + HL)

- **Gene mutation**: change in the nucleotide sequence of DNA
  - **Point mutations**: substitution, insertion, deletion
  - **Substitution types**: silent, missense, nonsense
  - **Frameshift mutations**: insertions or deletions shift the reading frame
- Causes of mutation: spontaneous errors, mutagens (UV radiation, chemicals, X-rays)
- Mutations as the source of genetic variation; most are neutral, some harmful, few beneficial
- **Sickle cell anemia**: single nucleotide substitution (GAG → GTG) → glutamic acid → valine in hemoglobin
- **Gene editing**: CRISPR-Cas9 system
  - Guide RNA directs Cas9 to specific DNA sequence
  - Cas9 cuts both DNA strands at target site
  - Cell repair mechanisms allow gene knockout or insertion
  - Applications: gene therapy, agriculture, research

## Key Terminology

| Term | Definition |
|------|-----------|
| **Semi-conservative replication** | Each new DNA molecule contains one original and one newly synthesized strand |
| **Transcription** | Synthesis of mRNA from a DNA template |
| **Translation** | Synthesis of a polypeptide from an mRNA template on ribosomes |
| **Codon** | Three-nucleotide sequence on mRNA that codes for an amino acid or stop signal |
| **Anticodon** | Three-nucleotide sequence on tRNA that is complementary to an mRNA codon |
| **Gene mutation** | Permanent change in the nucleotide sequence of a gene |
| **Frameshift mutation** | Insertion or deletion of nucleotides that shifts the reading frame of codons |
| **CRISPR-Cas9** | Gene-editing tool using guide RNA and Cas9 enzyme to cut DNA at specific sites |

## Common Misconceptions

1. **"Mutations are always harmful"** -- Most mutations are neutral; some are beneficial and drive evolution.
2. **"DNA replication is error-free"** -- DNA polymerase has a low but non-zero error rate; proofreading reduces but does not eliminate errors.
3. **"mRNA is a copy of the entire DNA"** -- mRNA is transcribed from specific genes, not the whole genome.
4. **"The genetic code varies between organisms"** -- The genetic code is nearly universal across all life, supporting common ancestry.
5. **"CRISPR can edit any gene perfectly"** -- Off-target effects remain a significant concern in CRISPR technology.
